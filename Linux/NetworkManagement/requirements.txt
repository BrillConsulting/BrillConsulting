# =============================================================================
# System Requirements (not installed via pip)
# =============================================================================

# These must be installed via system package manager:
#
# Ubuntu/Debian:
#   sudo apt-get install -y python3 iproute2 iptables traceroute
#   sudo apt-get install -y wireguard-tools openvpn  # Optional for VPN
#
# RHEL/CentOS:
#   sudo yum install -y python3 iproute iptables traceroute
#   sudo yum install -y wireguard-tools openvpn  # Optional for VPN
#
# Arch Linux:
#   sudo pacman -S python iproute2 iptables traceroute
#   sudo pacman -S wireguard-tools openvpn  # Optional for VPN
#
# Required System Tools:
# - python3 (3.8+)          : Core Python interpreter
# - ip (iproute2)           : Network interface configuration
# - iptables                : Firewall management
# - ss (iproute2)           : Socket statistics (netstat replacement)
#
# Optional System Tools:
# - traceroute              : Network path tracing
# - ping                    : ICMP echo testing
# - wg (wireguard-tools)    : WireGuard VPN management
# - openvpn                 : OpenVPN VPN server
# - netstat                 : Legacy network statistics (fallback)
# - mtr                     : Network diagnostic tool
# - nc (netcat)             : Network connectivity testing
