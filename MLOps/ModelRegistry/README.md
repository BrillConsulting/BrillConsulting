# Model Registry

Centralized repository for managing ML model lifecycle, versions, metadata, and deployment status with MLflow integration.

## Overview

Model registry provides central catalog of ML models, tracking versions, metadata, lineage, and deployment status for governance and reproducibility.

## Key Features

- Version control for models
- Lineage tracking (data + code + model)
- Deployment stage management (Dev/Staging/Prod)
- Model comparison across versions
- Access control and approvals

## MLflow Integration

Register models, transition stages, load by version or stage, track metadata and metrics.
