# Data Science with R Portfolio

Advanced statistical analysis and machine learning in R.

## Projects

### 1. Statistical Modeling
Comprehensive statistical analysis with multiple regression techniques.

**Features:**
- Linear regression with diagnostics
- Ridge/Lasso/Elastic Net regularization
- Logistic regression with ROC curves
- Polynomial regression
- Mixed effects models
- Generalized Additive Models (GAM)
- Model comparison (AIC, BIC, ANOVA)

**Tech Stack:** R, tidyverse, caret, MASS, lme4, mgcv

### 2. Time Series Analysis (Coming Soon)
ARIMA, SARIMA, and forecasting models

### 3. ML with caret (Coming Soon)
Machine learning workflows using caret framework

## Technologies

- **Core:** R 4.0+, tidyverse
- **Modeling:** caret, MASS, lme4, mgcv
- **Visualization:** ggplot2
- **Stats:** car, lmtest, glmnet

## Usage

```r
source("statistical_analysis.R")
example_analysis()
```
