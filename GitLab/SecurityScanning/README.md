# GitLab Security Scanning

SAST, DAST, dependency scanning, and container scanning integration.

## Features
- SAST (Static Application Security Testing)
- DAST (Dynamic Application Security Testing)
- Dependency scanning
- Container scanning
- License compliance
- Secret detection

## Technologies
- GitLab Security Scanning templates
- Trivy, Clair container scanners
- OWASP ZAP, Bandit, Semgrep
